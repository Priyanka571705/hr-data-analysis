# HR Data Analysis

Jupyter Notebook project for exploring HR data and practicing workforce analytics with Python.

## Category

Python, Data Analysis, HR Analytics

## What This Project Shows

- Exploratory analysis of HR-related data
- Data cleaning and feature inspection
- Department-level and employee-level summaries
- Visual analysis of workforce trends
- Practice turning raw data into readable observations

## Files

| File | Purpose |
| --- | --- |
| `HR_Data_Analysis_Project.ipynb` | Main analysis notebook |

## Tech Stack

`Python` `Jupyter Notebook` `Pandas` `NumPy` `Matplotlib` `Seaborn`

## How To Run

```bash
git clone https://github.com/Priyanka571705/hr-data-analysis.git
cd hr-data-analysis
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook HR_Data_Analysis_Project.ipynb
```

Or install from the included requirements file:

```bash
pip install -r requirements.txt
jupyter notebook HR_Data_Analysis_Project.ipynb
```
